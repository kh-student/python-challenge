## Instructions

### PyBank

1. clone repo
```
git clone git@github.com:kh-student/python-challenge.git
```

2. cd to assignment folder
```
cd ./python-challenge/PyBank
```

3. run python file from assignment folder (not from main folder)
```
python main.py
```
